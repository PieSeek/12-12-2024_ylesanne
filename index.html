LINUX MINT

Coming from Windows?

Linux Mint has the most familiar user interface to peaople coming from Windows. Linux Mint is also renowned for it's user friendliness - it doesn't require the user to be familiar with the terminal to do simple tasks suh as replacing the wallpaper or changing system font.

POP!_OS

On a laptop?

Pop!_OS is best for laptops. It uses the GNOME desktop environment and adds their own spin on it. It is very user-friendly and beacuse of its popularity, it has better support than samller distros.
